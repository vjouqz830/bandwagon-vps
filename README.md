一是机房迁移免费。买了 CN2 GIA-E 套餐，发现某个机房不合适，几步操作迁过去，不额外收费，停机时间很短。这在主机商里少见。

四是运营十几年了。2012 年到现在，踩雷的概率比新起步的商家低得多，掉链子和跑路这种事轮到搬瓦工的可能性很小。
